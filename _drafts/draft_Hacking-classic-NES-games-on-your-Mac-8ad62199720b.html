Hacking classic NES games on your Mac

This post was inspired by an awesome guy I know who never explained how to set this up to me.



sudo xcodebuild -license 
brew tap caskroom/cask
brew cask install java xquartz
brew install wine
brew install winetricks
winetricks settings fontsmooth=rgb
wine $FILE.exe


Now for the fun part, you can write Lua scripts to write bytes to memory addresses on any nes game you want. Want money? go crazy.
